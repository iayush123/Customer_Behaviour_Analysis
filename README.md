# Customer_Behaviour_Analysis
Its an project Based on Customer Behaviour Analysis to Increase saturation of customer for the brands


Project Summary

Analyzed 3,900+ customer transactions to identify patterns in purchasing behavior, customer segmentation, and revenue drivers.
Used Python, SQL, and Power BI to transform raw data into actionable business insights.

Tech Stack
Programming: Python (Pandas, NumPy)
Database: MySQL
Visualization: Power BI
Tools: Jupyter Notebook

Data Processing & Feature Engineering
Cleaned and validated dataset using Pandas
Handled missing values (median-based imputation)
Standardized column names (snake_case)
Created new features:
age_group
purchase_frequency_days

Key Insights
Male customers generated higher total revenue
Express shipping customers showed higher average spending
Top-rated products: Gloves, Sandals, Boots
Majority users fall under loyal customer segment
Young adults contribute the highest revenue
